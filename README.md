# Flappy Bird (Python/Pygame)

Clone of the classic Flappy Bird game built with Python and Pygame. The bird is controlled by clicks to navigate through pipes.

## Context
- **Date:** 2019
- **Type:** Personal Project

## Tech Stack
- Python 3
- Pygame

## How to Run
```bash
pip install pygame
python game_bird.py
```

## Files
- `game_bird.py` — Main game: bird physics, pipe generation, collision detection, scoring
- `imgs/` — Sprite assets: bird animation frames, pipes, background, base
